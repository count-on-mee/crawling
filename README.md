# Count On Me

## Count On Me 애플리케이션의 크롤링 소스 코드입니다.

### How to Install

<pre>
git clone https://github.com/count-on-mee/crawling.git
cd crawling
pip install beautifulsoup4
pip install webdriver-manager
</pre>

### How to Run

<pre>
python main.py
</pre>

### Structure

- `main.py`: 크롤링을 수행하는 메인 소스 코드 파일
